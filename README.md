# Temples of Tamil Nadu – Cultural Heritage Website

## 1. Title

Temples of Tamil Nadu – Cultural Heritage Website

---

## 2. Introduction

Tamil Nadu is one of the most culturally rich states in India, known for its magnificent temples, ancient architecture, classical dance forms, and vibrant traditions. The temples of Tamil Nadu are not only places of worship but also historical monuments that reflect Dravidian architecture, sculpture, music, and spiritual heritage.

This project focuses on designing and developing a responsive website titled “Temples of Tamil Nadu” that provides detailed information about famous temples, architectural styles, dance forms, and Tamil culture. The website includes multimedia elements such as images, audio, and video to enhance user engagement.

---

## 3. Objective

• To create an informative and interactive website about Tamil Nadu temples.  
• To showcase famous temples with detailed descriptions.  
• To highlight temple architecture, dance, and cultural heritage.  
• To implement multimedia elements like audio and video.  
• To design a responsive and user-friendly interface using HTML, CSS, and JavaScript.  

---

## 4. Existing System

Traditional information about temples is usually available through books, travel guides, or static web pages. These methods have certain limitations:

### Limitations:

- Lack of interactive content  
- Limited multimedia support  
- Static information without dynamic navigation  
- Not optimized for modern responsive devices  

---

## 5. Proposed System

The proposed system is a fully responsive and multimedia-enabled website that provides:

- Structured temple information  
- Interactive navigation between sections  
- Embedded audio and video elements  
- Smooth UI with attractive design  

The system integrates text, images, background music, and videos to provide an immersive cultural experience.

---

## 6. Features

### 1) Home Section

- Attractive landing page  
- Introduction to Tamil Nadu’s spiritual heritage  

### 2) Famous Temples Section

Includes detailed information about major temples such as:

1. Meenakshi Amman Temple  
2. Brihadeeswarar Temple  
3. Ramanathaswamy Temple  
4. Sri Ranganathaswamy Temple  
5. Arunachaleswarar Temple  
6. Chidambaram Nataraja Temple  
7. Ekambareswarar Temple  
8. Kailasanathar Temple  
9. Kapaleeshwarar Temple  
10. Srivilliputhur Andal Temple  

### 3) Architecture Section

• Explanation of Dravidian temple architecture  
• Information about Gopuram (temple towers), Mandapam, and Vimana  
• Sculptural carvings and stone structures  

### 4) Dance Section

- Information about classical dance form Bharatanatyam  
- Cultural importance and temple connection  
- Embedded dance performance video  

### 5) Culture Section

• Tamil traditions and festivals  
• Temple festivals like car festivals (Ther Thiruvizha)  
• Devotional music and rituals  

### 6) Multimedia Features

 • Background devotional audio  
 • Embedded cultural or temple videos  
 • High-quality temple images  
 • Attractive CSS styling and responsive layout

## 8. System Architecture

Temple Information Website Architecture

Presentation Layer:  
- Displays temple details, images, audio, and videos to users  

Application Layer:  
- Handles navigation, section switching, and interactive behavior using JavaScript  

Content Layer:  
- Stores temple information, cultural data, and multimedia resources  

Multimedia Layer:  
- Manages audio and video playback  

User Interaction Layer:  
- Allows users to navigate through Home, Famous Temples, Architecture, Dance, and Culture sections
